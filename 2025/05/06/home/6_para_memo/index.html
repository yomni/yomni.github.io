PARA 노트 정리법

  • 실행도를 기준으로 정리

1. Project

  • 끝이 명확함

  • 실행도가 가장 높음

  • 예시 : 자격증 합격하기, 시험공부, 블로그 글 작성, 제품 런칭, 전자책 출간, 이사하기

  • 이사하기 (project)

    • 부동산 목록
    • 가구 목록
    • 인테리어 정보
    • 부동산 계약 주의사항
    • …

  • 만약 프로젝트가 끝나면?

    • 영역 / 자원 / 아카이브 폴더로 이동
    • 예시) 이사하기 > 부동산 계약 주의사항 은 이사하기가 끝나면 자원이나 영역으로 이동

2. Area

  • 끝이 없음. 진행중
  • 나의 책임이나 역할이 있음
  • 예시 : 운동, 건강, 경제 계획, 직업, 가족
  • 프로젝트는 이 영역들 중 하나를 발전시키는 경우가 많다.

3. Resource

  • 내가 관심 있는 분야
  • 내 책임은 없음
  • 예시 : 음악, 커피, 미술, 역사, 뇌과학, 마케팅, 글쓰기, …
  • 영역과 자원의 차이는 개인의 책임
  • 자료는 아님

4. Archive

  • 실행도가 0임.
  • 종료된 프로젝트, 중단된 프로젝트
  • 책임이 사라진 영역, 관심이 사라진 자원
